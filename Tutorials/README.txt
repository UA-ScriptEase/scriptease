********************************************
*            Translator Tutorials          *
********************************************

These tutorials will help you familiarize 
yourself with how ScriptEase works and how 
to use it with the NeverWinter Nights game
engine, the Unity game engine, the Hack-E-bot 
Arduino environment, and others. 

LibraryTutorials: step-by-step walkthrough on
how to make your own causes, effects and 
definitions for the translator you're working
with. 

TranslatorInstructions: an indepth look at the
translator structure.
